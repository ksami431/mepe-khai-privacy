# Install Node.js - Action Required

## Quick Installation (Recommended)

### Option 1: Official Installer (Easiest - 3 minutes)

1. **Click this link**: https://nodejs.org/
2. **Download** the **LTS** version (green button on left)
3. **Run the installer** - it will be in your Downloads folder
4. **Click "Continue"** through all the prompts
5. **Enter your password** when asked
6. **Wait for installation** to complete

**Then come back and I'll continue the setup automatically!**

---

### Option 2: Homebrew (If you prefer command line)

**Open your Terminal app** and run these commands:

```bash
# Install Homebrew (you'll need to enter your password)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# After Homebrew installs, add it to your PATH
# (Follow the instructions Homebrew shows you)

# Then install Node.js
brew install node
```

## Why Node.js is Needed

Node.js provides:
- **npm** - Package manager to install React Native, Expo, etc.
- **JavaScript runtime** - To run the development server
- **Build tools** - To compile the app

Without it, none of the React Native tools will work.
