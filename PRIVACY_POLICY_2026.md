## 5. Data Security

### 5.1 Security Measures

We implement industry-standard security practices:

**Encryption:**
- All data transmitted between your device and our servers is encrypted using HTTPS/TLS
- Data at rest is encrypted in our database
- Passwords are hashed using bcrypt with salt

**Access Controls:**
- Row-level security ensures users can only access their own data
- Multi-factor authentication for administrative access
- Regular security audits and penetration testing

**Infrastructure Security:**
- Secure cloud hosting with Supabase
- Regular security updates and patches
- Automated backup systems
- DDoS protection

### 5.2 Data Retention

**Active Accounts:**
- We retain your data for as long as your account is active
- Data is necessary to provide our services

**Deleted Accounts:**
- Upon account deletion, all personal data is permanently deleted immediately
- No 30-day grace period for self-service deletion
- Anonymized usage statistics may be retained for analytics

**Legal Requirements:**
- Some data may be retained longer if required by law
- Financial records retained per regulatory requirements
